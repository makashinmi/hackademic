# Persistence

>Роль, ответственная за закрепление в системе и поддержание функционирования ПО до окончания работы.

Между доставкой ПО на целевую хост-машину и отправкой на C2 ключа, которым зашифрована вся система, может произойти что угодно. Твоя задача -- обеспечить успешное протекание этого процесса от начала до конца, несмотря на обстоятельства.


## Ожидаемые результаты

- Прерывание или маскировка рабочего процесса для противодействия обнаружению в диспетчере задач
- Восстановление работы между перезагрузками машины
- Деактивация программых средств защиты (диспетчер задач, антивирусы, дефендер)
- Отказ базовых мер восстановления (windows recovery points)
